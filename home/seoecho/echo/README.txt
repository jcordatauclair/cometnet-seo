Fonction du système Echo
--------------------------
Le système Echo calcule la probabilité de pertinence de nouveaux documents à partir de l'analyse d'anciens documents dont la pertinence a été déjà établie par un utilisateur. Il se présente comme un serveur basé sur les sockets (TCP-IP)

Entrée
-------
Le serveur echo récupère une chaine en entrée ayant la structure suivante :

iddocument jugement(0,1 ou 2) mots_du_document ;iddocument jugement mots_du_document ; iddocument jugement mots_du_document ;...
0 signifie non jugé
1 signifie pertinent
2 signifie non pertinent

Sortie
------
il retourne une chaine pour tous les documents non jugés entrés ayant le format suivant
iddocument:score:proba_pertinence;iddocument:score:proba_pertinence;iddocument:score:proba_pertinence;...

exemple :
 1070:0,84:0,40;1072:1,53:1,00;1073:3,46:1,00;1074:0,10:0,05;1075:1,59:1,00;1010:0,00:0,05;1012:0,64:0,05;1016:0,57:0,05;1017:0,28:0,05;1024:2,00:1,00;1026:0,30:0,05;1029:0,08:0,05;

Lancement
----------
Le serveur est lancé par la commande "java EchoServer"
Un exemple de client java (EchoClient) et php client.php sont donnés.
