===================================COMMANDS=====================================


EXEMPLES
--------------------------------------------------------------------------------
>> response.css('h1::text').extract()
   is the same as
>> response.xpath('//h1/text()').extract()
#  returns the value of the attribute "h1"

>> response.css('meta::attr(charset)'').extract()
   is the same as
>> response.xpath('//meta/@charset').extract()
#  returns the attribute "charset" of the tag "meta"
--------------------------------------------------------------------------------

================================================================================

META TAG #1 : TITLE
--------------------------------------------------------------------------------
/!\ : none

>> response.xpath('//title/text()').extract()
--------------------------------------------------------------------------------

META TAG #2 : DESCRIPTION
--------------------------------------------------------------------------------
/!\ : description or Description

>> response.xpath('//meta[@name="description"]/@content').extract()
>> response.xpath('//meta[@name="Description"]/@content').extract()
--------------------------------------------------------------------------------

META TAG #3 : H1, H2, H3, H4, H5, H6
--------------------------------------------------------------------------------
/!\ : syntax isn't always the same for hn tags
      | <h1 class="project-title"><span class="txt"><TEXT</span></h1>
      | <h1 class="project-title"><a href="txt">TEXT</a></h1>
      | <h1 class="project-title">TEXT</h1> ou <h1>TEXT<h1>

>> response.xpath('//h1/span/text()').extract()
   response.xpath('//h2/span/text()').extract()
   [...]
>> response.xpath('//h1/a/text()').extract()
   response.xpath('//h2/a/text()').extract()
   [...]
>> response.xpath('//h1/text()').extract()
   response.xpath('//h2/text()').extract()
   [...]
--------------------------------------------------------------------------------

META TAG #4 : KEYWORDS
--------------------------------------------------------------------------------
/!\ : keywords or Keywords

>> response.xpath('//meta[@name="keywords"]/@content').extract()
>> response.xpath('//meta[@name="Keywords"]/@content').extract()
--------------------------------------------------------------------------------
